# Social Media Strategy: GoCharting

## Content Pillars
1.  **The Educator (30%):** Deep-dives into Orderflow, CVD, and Market Profile.
2.  **The Builder (20%):** Showcasing Lipi Scripting capabilities and community scripts.
3.  **The Signal (40%):** Real-time "Liquidity Grab" and "Absorption" chart setups.
4.  **The Proof (10%):** User testimonials, "Why I Switched" stories, and broker integration updates.

## Weekly Posting Cadence (X & LinkedIn)
| Day | Content Type | Goal |
| :--- | :--- | :--- |
| **Mon** | "Market Prep" (Market Profile/CVD) | Awareness/Authority |
| **Tue** | "Lipi Script of the Week" | Differentiation |
| **Wed** | Technical Thread (Education) | Engagement/Saves |
| **Thu** | Feature Spotlight (e.g., DOM or TFR) | Feature Adoption |
| **Fri** | "Weekly Recap" / Community Charts | Proof/Community |

## Short-form Video (YouTube Shorts/Reels)
*   **Goal:** 2x per week.
*   **Format:** 45-60 seconds. "Spotting Absorption in 60 seconds."

## Key Conversion Tactics
*   **Link in Bio:** Linktree focusing on: 1. Try Pro 2. Join Discord 3. Download App.
*   **Twitter Threads:** Always end with a "Join our 2M+ trader community" CTA.
